### Projektin suorittaminen 
projektin voi ajaa omalla tietokoneella ensin asentamalla riippuvuudet poetryn avulla 
	*$ poetry install*
Käynnistäminen 
	*$ "poetry run invoke start"*
testien suorittaminen
	*$ poetry run invoke run-tests*
